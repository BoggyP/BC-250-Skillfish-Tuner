# Code of Conduct

Killfish Tuner is a community project.

Please keep discussions respectful, constructive, and focused on improving the software and the BC-250 ecosystem.

## Expected Behavior

- Be respectful
- Be patient with testers and contributors
- Provide useful details when reporting problems
- Accept that experimental hardware work can be difficult
- Credit community research and prior work

## Unacceptable Behavior

- Harassment
- Personal attacks
- Abusive language
- Deliberate misinformation
- Taking credit for others' work

Maintainers may moderate issues, pull requests, and discussions to keep the project productive.
