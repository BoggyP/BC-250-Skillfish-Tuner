# Skillfish Tuner 2.0

**Advanced BC-250 Management and Tuning Suite**

Skillfish Tuner is a modern Linux application developed specifically for the AMD BC-250 platform. It combines Compute Unit management, overclocking, fan control, governor integration, monitoring, diagnostics, and system management into a single, easy-to-use interface.

The project aims to bring together the tools, workflows, research, and community knowledge surrounding the BC-250 platform while providing a consistent and reliable user experience.

---

## Features

### Compute Unit Management

* View and manage CU topology
* Enable or disable selected Compute Units
* Persistent CU configuration across reboots
* Factory topology backup and restoration
* Live topology refresh
* Visual CU status overview

### Overclocking

* GPU frequency control
* Voltage tuning
* Real-time monitoring
* Persistent configuration support
* Built-in safety limits

### Fan Control

* Custom fan curves
* Fan calibration
* Automatic configuration saving
* Persistent fan settings across reboots
* Return-to-auto functionality

### Monitoring

* Real-time GPU telemetry
* Temperature monitoring
* Clock monitoring
* Voltage monitoring
* Fan RPM monitoring
* Service health monitoring

### System Management

* Governor integration
* Configuration backup and restore
* Hardware detection
* Diagnostic report generation
* Built-in logging

---

## Supported Platforms

Skillfish Tuner is designed primarily for:

* AMD BC-250
* Bazzite Linux
* Fedora-based distributions

Support for additional Linux distributions may be added in future releases.

## Installation

Download the latest release and extract the archive.

Run:

```bash
chmod +x Cyan-Skillfish-All-In-One.AppRun
./Cyan-Skillfish-All-In-One.AppRun
```

For the best experience, use the packaged installer or RPM release when available.
