# Killfish Tuner 2.0

**Advanced BC-250 Management and Tuning Suite**

Killfish Tuner is a modern Linux application developed specifically for the AMD BC-250 platform. It combines Compute Unit management, overclocking, fan control, governor integration, monitoring, diagnostics, and system management into a single interface for BC-250 enthusiasts.

The project aims to bring together community tools, workflows, research, and experimentation around the BC-250 while keeping the application clear, reliable, and suitable for advanced users.

## Short Description

Killfish Tuner is an advanced management and tuning suite for the AMD BC-250 platform. Control Compute Units, overclocking, fan behavior, governors, and hardware monitoring from a modern Linux interface designed specifically for BC-250 enthusiasts.

## Features

### Compute Unit Management

- View and manage CU topology
- Enable or disable selected Compute Units
- Persistent CU configuration across reboots
- Factory topology backup and restoration
- Last-good configuration restore
- Live topology refresh
- Visual CU status overview

### Overclocking

- GPU frequency control
- Voltage tuning
- Real-time monitoring
- Persistent configuration support
- Built-in safety limits

### Fan Control

- Custom fan curves
- Fan calibration
- Automatic configuration saving
- Persistent fan settings across reboots
- Return-to-auto functionality

### Monitoring

- Real-time GPU telemetry
- Temperature monitoring
- Clock monitoring
- Voltage monitoring
- Fan RPM monitoring
- Service health monitoring

### System Management

- Governor integration
- Configuration backup and restore
- Hardware detection
- Diagnostic report generation
- Built-in logging

## Supported Platforms

Killfish Tuner is designed primarily for:

- AMD BC-250
- Bazzite Linux
- Fedora-based distributions

Support for additional Linux distributions may be added in future releases.

## Installation

Download the latest release and extract the archive.

Run:

```bash
chmod +x Cyan-Skillfish-All-In-One.AppRun
./Cyan-Skillfish-All-In-One.AppRun
```

For the best experience, use the packaged installer or RPM release when available.

## Important Safety Notice

Killfish Tuner exposes advanced hardware controls. Overclocking, voltage modification, fan control, and Compute Unit configuration can affect stability, power consumption, thermals, and hardware lifespan.

Use this software at your own risk.
