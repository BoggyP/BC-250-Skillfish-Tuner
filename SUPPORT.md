# Support

For help with Killfish Tuner, please open a GitHub issue and include as much detail as possible.

## Useful Information

Please include:

- Killfish Tuner version
- Linux distribution
- Kernel version
- BC-250 board details, if known
- Current CU configuration
- Governor status
- Fan controller status
- Relevant logs

## Generate A Report

If the app can start, use the built-in report generator and attach the generated report archive to your issue.

Reports may include:

- Hardware information
- CU topology
- Governor status
- Service status
- Fan calibration
- System logs
- Application logs

Please review reports before uploading if you are concerned about sharing system information.
